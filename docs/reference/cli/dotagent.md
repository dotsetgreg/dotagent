# dotagent

## dotagent

Personal AI agent with Discord gateway, tools, memory, and provider routing

### Synopsis

dotagent is a lean, extensible agent runtime.

Use CLI commands to onboard, run local agent sessions, run the Discord gateway,
manage cron jobs, install skills, and manage executable toolpacks.

```text
dotagent [flags]
```

### Options

```text
  -h, --help      help for dotagent
  -v, --version   Show build/version metadata
```
