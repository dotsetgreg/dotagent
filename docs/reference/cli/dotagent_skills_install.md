# dotagent skills install

## dotagent skills install

Install a skill from GitHub

```text
dotagent skills install <owner/repo-or-path> [flags]
```

### Examples

```text
  dotagent skills install dotsetgreg/dotagent-skills/weather
```

### Options

```text
  -h, --help   help for install
```

### Options inherited from parent commands

```text
      --instance string   Instance ID under ~/.dotagent/instances (default "default")
```
