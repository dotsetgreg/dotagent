# dotagent toolpacks remove

## dotagent toolpacks remove

Remove an installed toolpack

```
dotagent toolpacks remove <id> [flags]
```

### Examples

```
  dotagent toolpacks remove github-cli
```

### Options

```
  -h, --help   help for remove
```
