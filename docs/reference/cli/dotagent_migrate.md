## dotagent migrate

Migrate legacy ~/.dotagent config/workspace into instance layout

```text
dotagent migrate [flags]
```

### Options

```text
      --force   Overwrite existing files in target instance when needed
  -h, --help    help for migrate
```

### Options inherited from parent commands

```text

```
